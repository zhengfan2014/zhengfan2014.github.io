音乐插件 jsososo

2020-05-15 1 min read # Jsososo # 网易云音乐 # QQ音乐 # Kodi插件

这是什么？

播放来自 母猪厂 和 马化腾家 的音乐，使用 Jsososo 的Api

插件说明

设置里api地址默认弄了jsososo的api服务器，不过你也可以搭建自己的母猪和马化腾api服务器。（如果有一天跑路的话）

详见Github：
NeteaseCloudMusicApi
QQMusicApi

食用说明

输入你的qq号和网易云的id可以获取公开歌单

输入自己的qq音乐cookie和网易云musicu可以获取自己的日推

关于歌词

我没有弄歌词功能，歌词是kodi的插件刮削来的（如果你有下载的话）。
如果歌词插件不能用的话，可以尝试到kodi官方论坛反馈（英文）
